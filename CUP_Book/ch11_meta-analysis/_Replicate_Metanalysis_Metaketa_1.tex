\section{Notes on replication script}\label{notes-on-replication-script}

\begin{itemize}
\item
  This .Rmd file produces all tables, and figures reported in Chapter 11
  of \emph{Information, Accountability, and Cumulative Learning: Lessons
  from Metaketa I} in the order in which they appear in the text.
\item
  The code exports results to folders ``tables'' and ``figures'' in this
  file's directory.
\item
  Replication options include:

  \begin{itemize}
  \item
    \texttt{prep}: if \texttt{TRUE}, generate data from raw files, if
    \texttt{FALSE}, loads cleaned data
  \item
    \texttt{n\_iter}: number of permutations of treatment assignment
    used in randomization inference analysis.
  \item
    \texttt{load\_iter\_matrix}: if \texttt{FALSE}, generate
    \texttt{n\_iter} columns of treatment assignment according to each
    study's assignment strategy. If \texttt{TRUE}, load existing
    assignment matrix with 2,000 iterations of treatment assignment.
  \item
    \texttt{ri\_action}: treatment effects and \emph{p}-values from
    randomization inference can be generated from scratch if set to
    \texttt{"generate"}, loaded from existing file if (\texttt{"load"}),
    or not returned (\texttt{"ignore"}) in results. Note that these
    estimations may take a few hours depending on the value of
    \texttt{n\_iter}.
  \item
    \texttt{run\_bayes}: if \texttt{TRUE}, will run Monte Carlo
    simulations. \texttt{run\_bayes\ \textless{}-\ FALSE} will skip the
    Bayesian analysis.
  \item
    \texttt{load\_specification\_matrices\ \textless{}-\ FALSE} will
    estimate, from scratch, treatment effects under all combinations of
    specification choices. Likewise,
    \texttt{load\_mde\_data\ \textless{}-\ FALSE} will re-run the
    diagnostic process for the minimal detectable effect analysis.
    Setting these options to \texttt{FALSE} will take several hours to
    run on an average machine.
  \item
    \texttt{full\_m1}: if \texttt{FALSE} turns off imputing \texttt{0}s
    for vote choice for cases with where voter turnout is 0 in the
    cleaned data.
  \end{itemize}
\item
  For a full clean replication empty the folders \texttt{figures} and
  \texttt{tables} and subfolder \texttt{data/temp} and set options:
  \texttt{prep\ \textless{}-\ TRUE},
  \texttt{ri\_action\ \textless{}-\ "generate"},
  \texttt{run\_bayes\ \textless{}-\ TRUE},
  \texttt{load\_iter\_matrix\ \textless{}-\ FALSE},
  \texttt{load\_specification\_matrices\ \textless{}-\ FALSE},
  \texttt{load\_mde\_data\ \textless{}-\ FALSE}.
\item
  This code requires installation of Rstan (unless
  \texttt{run\_bayes\ \textless{}-\ FALSE}). See installation
  instructions
  \href{https://github.com/stan-dev/rstan/wiki/RStan-Getting-Started}{here}.
\item
  The package \texttt{xlsx} requires a 64 bit Java install on PCs; for
  more detail see \url{https://java.com/en/download/win10.jsp}.
\end{itemize}
